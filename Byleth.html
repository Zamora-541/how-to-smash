Inicio Byleth

Byleth

Es el ultimo personaje del primer pase de peleadores. En su equipamiente tiene diferentes armas que usara en el combate, un arco (FailNaught), una lanza (Areadbhar), una hacha (AYMR) y finalmente una espada (Sword of the Creator).

Neutral B

Byleth usa el FailNaught para disparar una flecha para ganar ventaja a distancia

Side B

Byleth usa la lanza para ganar terreno y para que el enemigo no se acerque demasiado, tiene demasiado poder pero es facil cubrirse de este ataque

Down B

Es el ataque mas poderoso aunque te deja muy vulnerable, se debe usar para castigar cuando rompes un escudo

Up B

Este movimiento se usa para poder regresar al escenario o para agarra a personajes que estan encima en de ti
